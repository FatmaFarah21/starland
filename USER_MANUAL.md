## Getting Started

### System Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Stable internet connection
- Screen resolution of at least 1024x768 for optimal experience

### First Time Setup
1. Navigate to the system URL provided by your administrator
2. You will need a valid username and password to access the system
3. Contact your system administrator if you don't have login credentials

## Login and Authentication

### Logging In
1. Open your web browser and navigate to the system URL
2. Enter your username (usually your email address) and password
3. Click the "Login" button
4. You will be redirected to the appropriate dashboard based on your role

### User Roles
- **Admin**: Has access to all system features including user management
- **Management**: Can view dashboards, reports, and all data but cannot manage users
- **Entry**: Can only access data entry forms and limited viewing capabilities

### Password Management
- If you forget your password, click "Forgot Password" on the login screen
- You will receive an email with instructions to reset your password
- Contact your system administrator if you continue to have issues

## Data Entry

### Sales Entry
1. Navigate to the "Sales" section in the main menu
2. Fill in the following required fields:
   - Receipt Number (enter manually, no auto-generation)
   - Date (defaults to today's date)
   - Customer Name
   - Product Type (Drum, Jerry Can, Small Bottle)
   - Quantity
   - Unit Price (price per item)
3. Total Amount will calculate automatically based on quantity Ã— unit price
4. Select Payment Method (Cash, M-Pesa, Bank Transfer, Credit)
5. Add any notes if necessary
6. Click "Save Sale" to record the transaction

### Expense Tracking
1. Go to the "Expenses" section
2. Complete the required fields:
   - Expense Number (enter manually)
   - Date (defaults to today)
   - Category (select from dropdown: diesel, salary, maintenance, utilities, rent, other)
   - Vendor (if applicable)
   - Description
   - Amount
3. Choose Payment Method
4. Add notes if needed
5. Click "Save Expense" to record

### Diesel Purchases
1. Access the "Diesel" section
2. Fill in the required information:
   - Transaction Number (enter manually)
   - Date (defaults to today)
   - Supplier
   - Liters purchased
   - Unit Price per liter
3. Total Cost calculates automatically
4. Optionally add vehicle, driver, payment method, and notes
5. Click "Save Diesel Purchase"

### Repair Logs
1. Navigate to the "Repairs" section
2. Enter the following details:
   - Repair Number (enter manually)
   - Date (defaults to today)
   - Equipment being repaired
   - Type of repair
   - Problem description
   - Cost of repair
3. Add repair shop, technician, payment method, and notes as needed
4. Click "Save Repair"

### Damage Reports
1. Go to the "Damages" section
2. Complete the required fields:
   - Damage Number (enter manually)
   - Date (defaults to today)
   - Item Type
   - Item Name
   - Cause of damage
   - Estimated Value
   - Severity (Low, Medium, High, Critical)
3. Optionally add responsible party and description
4. Click "Save Damage Report"

## Management Interface

### Dashboard Overview
The main dashboard provides an at-a-glance view of:
- Today's sales revenue
- Monthly sales revenue
- Total monthly expenses
- Net profit for the month
- Recent activity feed

### Navigation
- Use the left sidebar to navigate between different sections
- The dashboard provides the highest-level overview
- Each specific section (Sales, Expenses, etc.) provides detailed views

### Filtering and Searching
- Use the date range selector to view data for specific periods
- Use the search bar in each section to find specific records
- Sort data by clicking on column headers in tables

## Reporting

### Report Types
The system offers various reports:
- Sales Reports (daily, weekly, monthly)
- Expense Analysis
- Diesel Consumption Reports
- Repair Summary
- Damage Reports
- Cash Flow Statements
- Inventory Reports
- Customer Trend Analysis

### Generating Reports
1. Go to the "Reports" section
2. Click on the desired report type
3. The system will generate a CSV file automatically
4. The file will download to your computer
5. Open with Excel or any spreadsheet application for analysis

### CSV File Format
- All reports export as CSV (Comma-Separated Values) files
- Files are compatible with Microsoft Excel, Google Sheets, and other spreadsheet software
- Column headers are descriptive for easy understanding
- Date formats follow YYYY-MM-DD standard

## Troubleshooting

### Common Issues

#### Login Problems
- **Issue**: Cannot log in
- **Solution**: 
  - Double-check your username and password
  - Ensure Caps Lock is off
  - Try resetting your password using the "Forgot Password" link
  - Contact your system administrator if problems persist

#### Data Not Saving
- **Issue**: Data entry forms don't save
- **Solution**:
  - Check your internet connection
  - Ensure all required fields are filled
  - Look for error messages on the form
  - Refresh the page and try again

#### Reports Not Generating
- **Issue**: "Error generating report" message
- **Solution**:
  - Check internet connection
  - Ensure you have the correct permissions
  - Try again after a few moments
  - Contact your system administrator if the issue continues

### Performance Tips
- Close unused browser tabs to improve performance
- Clear browser cache periodically
- Use the latest version of your preferred browser
- Ensure a stable internet connection for best experience

### Getting Help
- Contact your system administrator for user access issues
- Check the system status with your IT department
- For technical issues, provide error messages and screenshots when requesting help
- Keep your login credentials secure and never share with others

## Best Practices

### Data Entry
- Enter data promptly after transactions occur
- Double-check numbers and amounts before saving
- Use descriptive notes for unusual transactions
- Enter receipt numbers exactly as they appear on physical receipts

### Security
- Log out when leaving your workstation
- Never save passwords in browsers on shared computers
- Report suspicious activity to your administrator
- Keep your login credentials confidential

### Regular Tasks
- Review daily sales and expenses regularly
- Reconcile entries with physical records periodically
- Generate and review reports on a schedule
- Back up important data as per company policy
